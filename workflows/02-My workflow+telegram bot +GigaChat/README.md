# Мой второй workflow на n8n с подключением модуля GigaChat для ответов пользователя на вопросы полученные с телеграмм бота

## Проблема
Базово нужно разобраться как подключить телеграмм бота, передать вопрос пользователя от бота в модуль гигаЧата и вернуть обратно ответ

## Решение
Создал телеграмм бота, взял идентификатор и прописал в первый модуль флоу telegram trigger, который ожидает сообщени от пользователя и только в случае его получения n8n запускает процесс дальше и отправляет текст сообщения в модуль  GigaChat. После этого ответ от ИИ перенаправляется в бота обратно посредством модуля send a text message.
В процессе возникли первые проблемы и вопросы - в частности для соединения телеграмма и n8n локальный адрес (в докере развернутый   n8n именно по нему) не соединяется, есть необходимость выдать открытый адрес, временно для обхода пришлось установить ngrok командами brew install ngrok (для мака на чипе м), сделать конфигурацию ngrok config add-authtoken ....,
затем запустить прокидывание порта ngrok http 5678 (пока без впн не работает, так что немного танцуем с бубнами). Нужно обратить внимание что грок выдает адрес (который как я понял может меняться), который вам нужно прописать в модуле тригера в n8n в Webhook URLs, это важно, для соединения телеграмм бота и локально развернутого n8n. В дальнейшем я буду улучшать архитектуру и разсернутое окружение. Как памятка, ngrok считается не очень "безопасным", более правильное решение, которое я позже попробую, использовать nginx.

## Архитектура
телеграмм бот обращается к адресу выданному гроком в локально развернутый на докере n8n, запрос обрабатывается ИИ и ответ возвращается обратно.

## Что внутри

- описание работы в readme

## Как использовать
1.Предусловия - локально развернутый и запущенный n8n 
2. Установите Community Node для GigaChat
3. Настройте credentials
4. Импортируйте workflow.json
5. прокиньте порт (к примеру гроком или чем-то другим)
6. создайте телеграмм бота и используйте айди
7. запустите флоу и отправьте сообщение в бота, при успешном запуске вы получите ответ

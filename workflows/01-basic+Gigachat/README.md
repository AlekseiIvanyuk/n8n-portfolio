# Мой первый workflow на n8n с подключением модуля GigaChat для ответов пользователя

## Проблема
Базово нужно разобраться как подключить сторонний модуль в виде GigaChat и запускать флоу

## Решение
Как n8n + GigaChat работают в совокупности

## Архитектура
Пока здесь пусто, простой флоу

## Что внутри
- запуск по нажатию
- Set который задает значения поля questions и передает его в модуль AI
- модуль GigaChat который принимает на вход вопрос и отдает на выход ответ

## Как использовать
1. Установите Community Node для GigaChat
2. Настройте credentials
3. Импортируйте workflow.json
4. Напишите в блоке set свой вопрос и запустите флоу


## Чему научился в процессе
Подключать AI модули задавать простые параметры и запускать флоу. Инфраструктурно - развернуть на localhost n8n внутри докера, загрузить итоговый json флоу в локальный репозиторий, затем запушить в гитхаб, для сбора проектов в портфолио
